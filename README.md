## Descrição

Bem-vindo ao Desafio Frontend Carbel, uma proposta interativa e visualmente atraente que busca destacar minhas habilidades em HTML, CSS, JavaScript, jQuery, Gulp e Bootstrap. Este projeto foi desenvolvido como parte de um processo seletivo para a vaga de Frontend na Carbel, uma concessionária de veículos.

## Como iniciar o projeto

- **Clone o repositório:**
```bash
git clone https://github.com/igorlolyan/desafio_carbel_alpes.git
```
- **Abra o diretório do projeto:**
```bash
cd repository-name
```
- **Instalar dependências do projeto:**
```bash
npm install
```
- **Start:**
```bash
npm start
```
- **Build:**
```bash
npm run build
```
